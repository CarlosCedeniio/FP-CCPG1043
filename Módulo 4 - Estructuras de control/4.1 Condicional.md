## Tomando Decisiones con `if`, `elif` y `else`

A veces, queremos que un bloque de c√≥digo se ejecute **solo si** se cumple una cierta condici√≥n.

### La declaraci√≥n `if`
La estructura m√°s simple. Ejecuta el c√≥digo que contiene solo si la condici√≥n es `True`.

**Sintaxis:**
```python
if condicion:
    # Este bloque se ejecuta si la condicion es verdadera (True)
    print("La condici√≥n se cumpli√≥.")
```

---

**Ejemplo:**
```python

if edad >= 18:
    print("Eres mayor de edad.")
```

```python

if nombre == "Carlos":
    print("Te llamas Carlos")
```

---

### A√±adiendo un plan B con `else`
Se usa para ejecutar un bloque de c√≥digo si la condici√≥n del `if` resulta ser `False`.

**Sintaxis:**
```python

    # Bloque si es True
    print("La condici√≥n se cumpli√≥.")
else:
    # Bloque si es False
    print("La condici√≥n NO se cumpli√≥.")
```

---

**Ejemplo:**
```python

    print("¬°Qu√© calor! Es un buen d√≠a para ir a la playa.")

    print("Mejor me abrigo un poco.")
```

### M√∫ltiples condiciones con `elif`
Cuando tienes m√°s de dos posibilidades, `elif` (contracci√≥n de "else if") te permite a√±adir tantas condiciones intermedias como necesites. Python las evaluar√° en orden y ejecutar√° **solo el primer bloque** cuya condici√≥n sea `True`.

**Sintaxis:**
```python

    # Bloque 1
elif condicion_2:
    # Bloque 2
else:
    # Bloque si ninguna anterior fue True
```

---


**Ejemplo:** Calificar una nota.
```python

    calificacion = "B"

    calificacion = "C"

```
